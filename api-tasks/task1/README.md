# Task 1 — Fetch & Print Users

## Objective
Fetch users from JSONPlaceholder API and print:
- Name
- Email
- City

## API Used
https://jsonplaceholder.typicode.com/users

## Steps
1. Send GET request
2. Check status code
3. Convert response to JSON
4. Loop through users
5. Print name, email, city

## Run

python fetch_users.py

## Output Example

Name: Leanne Graham
Email: Sincere@april.biz
City: Gwenborough
